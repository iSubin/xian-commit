## 项目定位

xian-commit 是面向 Codex、Claude Code 等 AI Coding Agent 的工程级 Git 交付治理 Skill。它通过项目级 Agent 资产、可配置 policy 和四个原生 Git hooks，约束 staged path、commit message 与 push 边界。

项目不是常驻服务，不提供 user 级或全局安装，也不替代业务测试、代码 review、服务端分支保护或外部 hook 管理器。
