## 启动与健康检查

xian-commit 没有后端、前端、数据库或健康端点。最小安装 smoke：

```sh
tmp_repo=$(mktemp -d)
git -C "$tmp_repo" init
cd "$tmp_repo"
sh /path/to/xian-commit/install.sh
sh /path/to/xian-commit/install.sh verify
```
